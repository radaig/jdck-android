# 获取京东cookie程序

## 说明

本程序只是简化了打开手机版网页、输入手机号、获取cookie步骤，并不能一键扫码登录获取。鉴于PC网页验证码老是错误，于是开发了安卓版。

## 功能
* 保存/删除手机号
* 一键输入手机号并打勾协议
* 一键复制cookie
* 清空cookie并重新加载
* 支持青龙面板登录及更新cookie
* 更新了使用青龙面板的，open/env进行ck的更新，而不必使用admin账号。
